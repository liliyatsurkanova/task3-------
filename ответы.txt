1. Какими способами можно сделать горизонтальное выравнивание по центру? Минимум 3 варианта, можно больше
Не совсем поняла вопрос. При помощи свойств justify-items и justify-self можно выравнить элементы по строке (по центру - значение center), а при помощи align-items и align-self можно выравнить по колонкам относительно границ ячейки (по центру - значение center). 
2. Что за единица измерения - fr?
fraction - фракция -  единица доступного пространства, которое можно разделить на сколько угодно частей. 
3. Как можно задать грид с 5 колонками шириной по 20%? Минимум 2 способа.
1 способ: grid-template-columns: repeat(5, 20%)
2 способ: grid-template-columns: 20% 20% 20% 20% 20%
4. Самостоятельно разберитесь, что такое auto-fill и auto-fit ?
auto-fill - это значение, которое пытается поместить наибольшее количество повторяющихся элементов в колонку, сколько их может уместиться без переполнения. Когда места для добавления другого элемента не хватает, следующий элемент будет размещён на следующей строке, а это место, которое он не смог заполнить, останется пустым.
auto-fit - это значение, которое ведёт себя почти как auto-fill, за исключением того, что любая пустая область будет схлопываться и растягивать элементы в этой строке – напоминая поведение флексбоксов, когда по мере уменьшения доступного пространства колонки схлопываются.

7. Что такое и как задается grid area?
grid area — это область внутри сетки, определенная с помощью комбинации ячеек, рядов и столбцов.
задается: grid area: значение grid-row-start/ значение grid-column-start/ значение grid-row-end/ значение grid-column-end;
8. Приведите пример использования grid-template-areas
.item-a {
    grid-area: header;
}
.item-b {
    grid-area: main;
}
.item-c {
    grid-area: sidebar;
}
.item-d {
    grid-area: footer;
}

.container {
    grid-template-columns: 50px 50px 50px 50px;
    grid-template-rows: auto;
    grid-template-areas:
        'header header header header'
        'main main . sidebar'
        'footer footer footer footer';
}
9. Каким свойством можно задать такое поведение элементов?

10. Самостоятельно разберись, как работают именованные линии? Есть ли какие-то рекомендованные правила наименований? Если да, то какие?
Имена линиям можно задать внутри квадратных скобок в списке треков. Можно использовать почти любое имя (кроме ключевого слова span), однако можно использовать и область, созданную из этих линий, если их имена заканчиваются на -start и -end. Линии могут иметь несколько имён, заданных внутри квадратных скобок и разделённых пробелом. Несколько линий могут иметь одинаковое имя; если нужна определённая, просто добавьте после названия номер нужной линии по порядку.
12. Изучи материалы и найдите, для каких ситуаций подходит position: fixed?
Когда нужно, чтобы блок при прокрутке страницы зафиксировался на одном месте.
